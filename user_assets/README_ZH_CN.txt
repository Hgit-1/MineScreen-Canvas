MineScreen Canvas 自定义 UI 素材上传目录
========================================

这里是正式 build 前的可选图片投放目录。请不要修改文件名：

1. panel_decoration.png
   - 用于配置面板/主机面板的可选角落装饰
   - PNG，推荐 256x256，透明背景
   - 最大 1024x1024
   - 主体建议靠右上或右下，不要包含需要精确阅读的小字
   - 不透明白底会覆盖控件，必须先去除背景
   - Mod会裁剪透明边界，等比缩放到面板右下安全区，并以默认55%不透明度绘制

2. loading_decoration.png
   - 用于 WEB 加载页/错误页的可选透明装饰
   - PNG，推荐 512x512 或 1024x1024，透明背景
   - 最大 2048x2048
   - 请为中央加载动画和错误文字留出空间
   - Mod保持原始比例，并叠加在页面缩略图或默认加载背景上

两张图片均为可选；不上传时 MineScreen 使用纯简约 UI，不显示占位图或紫黑缺失纹理。
图片只会作为本 Mod 的本地资源打包，不会从网络下载，也不会上传到服务器。

放好图片后请通知 Codex；届时会检查尺寸、接入渲染并执行最终 clean build。
